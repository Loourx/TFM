04 — Sistema B: Separadores de sección isométricos
Cómo usarlos: Cada separador se genera con Imagen 3 usando el prompt master (prefijo fijo) + el prompt específico de cada sección. El título y número se superponen en InDesign sobre la imagen generada. Paleta isométrica: grises + acento #00C9B1 + #1A5FFF en UN único elemento por separador.

05 — Prompt master Imagen 3 (prefijo fijo)
PREFIJO — copiar siempre íntegro, nunca modificar:

Isometric illustration, minimal design, architectural style, no text, no labels, no people, clean lines, flat lighting, F2F2F2 (#F2F2F2) background, gray palette (#C8C8C8 buildings, #1C1C1C roads), single accent color #1A5FFF (electric blue) used sparingly on ONE element only, square 1:1 format, white space preserved, 2026 transport infrastructure theme —

SUFIJO — específico por sección (ejemplo Sección 01):
city crossroads at night, empty streets, dark asphalt, white road markings, one traffic light glowing blue, no cars, flat gray buildings in background, very minimal
Regla de uso: Nunca cambiar el prefijo entre sesiones. Si Imagen 3 genera algo inconsistente, regenerar con el mismo prompt antes de modificar ningún parámetro. La consistencia viene del prefijo bloqueado, no del sufijo.
06 — Cómo solicitar gráficos a Claude
Formato de request: "Claude, genera un [TIPO_CHART] con los siguientes datos: [datos]. Vertical [A/B/C]."

Ejemplo: "Claude, genera un BAR_HORIZONTAL con cobertura nocturna: Madrid 40%, Barcelona 55%, Berlín 80%, París 72%, Londres 90%. Vertical A. Fuente: pendiente."

Output: Artefacto React (.jsx) con los tokens aplicados, exportable como imagen estática para InDesign.
